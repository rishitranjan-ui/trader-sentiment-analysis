# Trader Performance vs Market Sentiment Analysis

## Objective
Analyze how market sentiment (Fear/Greed) impacts trader behavior and performance.

## Datasets
- Bitcoin Fear & Greed Index
- Historical Trader Data (Hyperliquid)

## Methodology
- Data cleaning and preprocessing
- Date alignment and merging datasets
- Computation of key metrics (PnL, win rate, trade frequency)
- Comparative analysis based on sentiment

## Key Insights
1. Greed phases show higher profitability but increased risk-taking behavior.
2. Fear phases lead to reduced trading activity and lower returns.
3. High-volume traders outperform during Greed phases, indicating experience-driven gains.

## Strategy Recommendations
- Reduce position size and leverage during Fear periods.
- Avoid overtrading during Greed phases.
- High-volume traders can selectively increase exposure in bullish sentiment.

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
